üîå API Reference

MCP (Model Context Protocol)

Nullblock uses the MCP standard for AI agent integration:

Available Tools

  • get_market_data - Real-time market data
  • execute_trade - Execute trading orders
  • get_portfolio - Portfolio information
  • social_sentiment - Social media sentiment analysis
  • arbitrage_opportunities - Find arbitrage opportunities

Example Usage

# Get real-time market data result = await session.call_tool("get_market_data", { "symbol": "ETH/USD", "exchange": "binance" }) # Execute a trade result = await session.call_tool("execute_trade", { "symbol": "ETH/USD", "side": "buy", "amount": 1.5, "price": 2450.00 })

REST API Endpoints

Base URL: https://api.nullblock.io/v1 GET /market/price/{symbol} GET /market/volume/{symbol} POST /trade/order GET /portfolio/{wallet_address}

üö® Troubleshooting

Common Issues

Wallet Connection Fails

  • Ensure wallet extension is installed and unlocked
  • Check browser console for error messages
  • Try refreshing the page and reconnecting

MCP Connection Issues

  • Verify MCP server is running on correct port
  • Check firewall settings
  • Ensure Python dependencies are installed

Trading Features Not Available

  • Confirm wallet is connected
  • Check if you're on the correct network
  • Verify sufficient balance for trading
